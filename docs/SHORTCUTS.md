# Keyboard Shortcuts & Productivity Features

## Beginners

| Shortcut | Action |
|----------|--------|
| **Ctrl+K** | Open command palette — search any page or action |
| **Ctrl+/** | Open quick start guide (6-step demo walkthrough) |
| **Shift+?** | Show all keyboard shortcuts |
| **Ctrl+Shift+1** | Switch to Manager (Sarah) for approvals |
| **Ctrl+Shift+2** | Switch to Developer (Alex) for voting |
| **Ctrl+Shift+3** | Switch to Intern (Sam) — memory governance demo |
| **Esc** | Close palette, modal, or guide |
| **Ctrl+Shift+R** | Refresh current page data |

**UI:** Sidebar has **Ctrl+K**, **Quick start** (book icon), and **?** buttons. A floating **Quick start guide** chip appears for first-time users.

## Everyone

| Shortcut | Action |
|----------|--------|
| **Ctrl+Shift+T** | Toggle dark / light mode |
| **Ctrl+B** | Collapse / expand sidebar |

## Experts (vim-style navigation)

Press **G**, then within 1.2 seconds:

| Key | Destination |
|-----|-------------|
| **D** | Dashboard |
| **L** | Lifecycle OS |
| **F** | Feature Packs |
| **P** | Suggestions |
| **B** | Branches |
| **T** | Tasks |
| **M** | Memory Governance |
| **G** | Project Brain |

## Command palette actions

- Navigate to any page
- Toggle theme
- Switch demo roles
- Refresh data
- Open shortcuts or quick start guide

## Footer hint bar

Bottom of screen shows active shortcuts; displays **Go to…** when **G** is pressed.
